## Getting Started

### Prerequisites

You need first to install the modules using npm command:
* npm
  ```sh
  cd client
  npm install
  ```
  ```sh
  cd server
  npm install
  ```
  

### Installation



1. Clone the repo
   ```sh
   git clone https://github.com/OssamaLouati/EventScroller-PFA.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run the local server in `Terminal`
   ```cmd
   cd client
   npm start
   ```
   ```cmd
   cd server
   nodemon server
   ```
